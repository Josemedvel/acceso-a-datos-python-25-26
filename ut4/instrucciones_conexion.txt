Importante para conectar con la base de datos:
- instalar psycopg2 con pip
- Crear una cuenta en supabase.com
- Crear un proyecto nuevo
- clicar en Connect y debería salir "Connection string"
- Clicar Method: Session pooler
- Crear un archivo .env en la carpeta ut4 con las variables de entorno:
    SUPABASE_DB_HOST= debería ser algo como aws-1-eu-west-1.pooler.supabase.com
    SUPABASE_DB_NAME= postgres
    SUPABASE_DB_USER= debería ser postgres.[aquí va el id del proyecto]
    SUPABASE_DB_PASSWORD= la contraseña que hayas puesto al crear el proyecto
    SUPABASE_DB_PORT= 5432
